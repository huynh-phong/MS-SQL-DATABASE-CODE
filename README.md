## 📌 Introduction
This repository contains practice exercises for the **Database / SQL Server** course, covering **LAB 1 → LAB 2**.  
It includes **SQL scripts** for basic database operations, queries, and problem solving learned in class.

---

## 📚 Content

### 🔹 Lab 1 - Basic SQL & Table Operations
- Create and manage tables  
- Insert, update, delete, and query data  
- Simple SELECT queries with filtering, sorting, and basic aggregate functions  

### 🔹 Lab 2 - Advanced Queries & Data Manipulation
- Join tables (INNER JOIN, LEFT JOIN, RIGHT JOIN)  
- Grouping, aggregation, and HAVING clause  
- Subqueries and nested queries  
- Simple stored procedures and functions (if applicable)
